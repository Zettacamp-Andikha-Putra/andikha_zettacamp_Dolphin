Saat ini kesibukan saya adalah sedang menjalani sedang menjalani bootcamp dari Zettabyte bernama Zettacamp. Saat ini sudah masuk ke minggu kedua bootcamp. Untuk di minggu ini, materi yang dibawakan mengenai CSS Cascading Style Sheet. CSS adalah bahasa Cascading Style Sheet dan biasanya digunakan untuk mengatur tampilan elemen yang tertulis dalam bahasa markup, seperti HTML. CSS berfungsi untuk memisahkan konten dari tampilan visualnya di situs.
